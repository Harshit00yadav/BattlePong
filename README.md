# BattlePong
a simple ping pong game

## Installation :
 + clone the repository
 + ```git clone https://github.com/Harshit00yadav/BattlePong.git```

## Requirements :
 + Java installed on the system.

## How to run :
 + go into the directory
 + ```cd BattlePong```
 + run the bash file
 + ```bash BattlePong.sh```

## Controlls :
 + w -> move up
 + s -> move down
